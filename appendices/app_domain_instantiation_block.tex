\subsection{Domain Instantiation Across Validated Domains}
\label{sec:domain-instantiation-\thechapter}

The theorem above is stated for an abstract CPS with observation vector
$z_t$, reliability score $w_t$, state uncertainty region
$C_t^{\text{RAC}}$, feasible action set $\mathcal{A}_t$, and violation
predicate $v(\cdot)$.  Table~\ref{tab:domain-instantiation-\thechapter}
maps these objects to their concrete counterparts in each of the five
forecasting-capable validated domains.  Each column instantiates the shared
adapter contract; no domain-specific assumption propagates to the abstract
theorem statement.
