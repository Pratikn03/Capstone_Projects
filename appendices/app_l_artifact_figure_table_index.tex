\section{Additional Artifacts}

\begin{itemize}
\item \texttt{baseline\_lock\_manifest.json}, \texttt{final\_lock\_manifest.json} --- Lock manifests for reproducibility.
\item \texttt{conference\_figure\_inventory.json} --- Figure-to-file mapping for paper.
\item \texttt{deployment\_evidence\_manifest.json}, \texttt{deployment\_evidence\_map.json} --- Deployment evidence linkage.
\item \texttt{stats\_summary.json} --- Aggregate statistics for paper metrics.
\end{itemize}

All paths are relative to the repository root. The \texttt{reports/publication/sweeps/} subdirectory contains per-run outputs (e.g., \texttt{runs/alpha0=0.05\_\_ph\_lambda=3\_\_kpen=0.3/}) with \texttt{dc3s\_main\_table.csv}, \texttt{dc3s\_fault\_breakdown.csv}, \texttt{calibration\_plot.png}, and \texttt{violation\_vs\_cost\_curve.png} for each grid point.
