\chapter*{Acknowledgments}
\addcontentsline{toc}{chapter}{Acknowledgments}

This thesis exists as part of a larger research effort centred on certificate-aware safety in
physical AI systems.  The battery-domain work was chosen as the first reference domain
because it provides a concrete, interpretable, and reproducible surface on which the hidden
gap between observed-state safety and true-state safety can be measured clearly.

The broader ORIUS research vision extends beyond batteries into benchmark design, temporal
certificate validity, graceful degradation, composition across interacting agents, and runtime
deployment support.  This project file is structured so those future directions can be appended
cleanly rather than rewritten from scratch.
