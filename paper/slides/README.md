# ORIUS Slide Decks

- `orius_universal_first_defense.tex`: technical defense deck for the full universal-first monograph.
- `orius_public_summary.tex`: shorter public-facing summary deck.

Compile from the repository root:

```bash
cd paper/slides
pdflatex -interaction=nonstopmode -halt-on-error orius_universal_first_defense.tex
pdflatex -interaction=nonstopmode -halt-on-error orius_public_summary.tex
```
