# Data Card: EIA-930 United States Balancing-Authority Surface

## 1. Dataset name and version
- Name: EIA-930 hourly operations data
- Regions used in repo: `US_MISO`, `US_PJM`, `US_ERCOT`

## 2. Source URL and download
- Source URL: [https://www.eia.gov/opendata/browser/electricity/rto](https://www.eia.gov/opendata/browser/electricity/rto)
- Repo-local processed paths:
  - `data/processed/us_eia930/features.parquet`
  - `data/processed/us_eia930_pjm/features.parquet`
  - `data/processed/us_eia930_ercot/features.parquet`

## 3. License
- U.S. government public data terms

## 4. Row count
- Locked publication rows:
  - `US_MISO`: 13,638
  - `US_PJM`: 13,639
  - `US_ERCOT`: 13,453

## 5. Feature schema
- Core signals: demand/load, wind generation, solar generation
- Weather joins from Open-Meteo
- Region-specific calendar and lag features
- Feature counts range from 62 to 114 by region

## 6. Temporal extent
- Locked evaluation surface: 2024-07 through 2026-01

## 7. Split policy
- Time-ordered train, calibration, validation, and test surfaces
- Region-specific processed outputs remain isolated to avoid leakage

## 8. Preprocessing pipeline
- EIA-930 extraction
- Regional filtering and timestamp normalization
- Weather and calendar augmentation
- Lag/rolling feature engineering
- Export to region-specific processed parquet files

## 9. Known issues
- Regional differences in signal completeness and feature count
- Price surface is proxy-based for some U.S. rows

## 10. Intended ORIUS use
- U.S. battery transfer and multi-region validation surface
- Cross-region stress testing for calibration and replay generalization

## 11. Citation
- See `paper/bibliography/orius_monograph.bib` for EIA and transfer-stress references.
